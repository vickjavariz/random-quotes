## About the Project

[![Random Quotes Screenshot][randomquotes-screenshot]](https://randomquotes.vercel.app/)

The main functionality of this web application is to display random quotes. Through its simple interface, the user can load a new randomly selected quote by clicking on the `New Quote` button. If the user wishes to share the quote, they can easily do so via Twitter using the button sporting the Twitter logo.

### Built With

- [React](https://reactjs.org/)
- [Styled Components](https://styled-components.com/)
- [Axios](https://axios-http.com/)
